## 一、项目概述

### 1.1 项目目标

将分散的 NiceGUI 工具项目聚合到统一平台 **EncyHub**，采用现代前端技术栈（React + Tailwind + shadcn/ui），便于 AI 辅助开发和维护。

### 1.2 待聚合工具

| 工具 | 原路径 | 核心功能 |
|------|--------|----------|
| AdbMaster | `E:\Such_Proj\Other\AdbMaster` | Android 设备管理 |
| FlowSvn | `E:\Such_Proj\Python\AutoSvn` | SVN 定时更新 + 触发器 |
| GmConsole | `E:\Such_Proj\Python\HaruRuntimeGMClient` | 游戏 GM 控制台 |

### 1.3 核心需求确认

| 项目 | 决策 |
|------|------|
| 运行模式 | 纯 Web（浏览器访问），支持内网其他设备访问 |
| 工具启动 | 记住上次状态，有 start 脚本 |
| 打包策略 | 平台不打包，各工具保留独立 build.py |
| 虚拟环境 | 共用一个（uv 管理） |
| 数据目录 | `EncyHub/data/{工具名}/` |
| 热重启 | 支持单个工具重启，不影响平台 |
| 扩展性 | 支持后续添加新工具，读源码即可迁移 |
| 登录认证 | 不需要 |

### 4.2 进程管理 (process_manager.py)

```python
class ProcessManager:
    async def start_tool(self, tool_id: str) -> bool:
        """启动工具子进程"""
        port = find_free_port()
        process = subprocess.Popen(
            ["uv", "run", f"tools/{tool_id}/main.py"],
            env={**os.environ, "PORT": str(port)},
            cwd=ROOT_DIR
        )
        # 更新注册表
        registry.update(tool_id, pid=process.pid, port=port, enabled=True)
        return True

    async def stop_tool(self, tool_id: str) -> bool:
        """停止工具子进程（递归杀死子进程树）"""
        pid = registry.get(tool_id).pid
        parent = psutil.Process(pid)
        for child in parent.children(recursive=True):
            child.kill()
        parent.kill()
        registry.update(tool_id, pid=None, port=None)
        return True

    async def restart_tool(self, tool_id: str) -> bool:
        """热重启工具"""
        await self.stop_tool(tool_id)
        await self.start_tool(tool_id)
        return True
```

### 4.4 请求代理

EncyHub 主进程代理请求到各工具子进程：

```python
@app.api_route("/api/{tool_id}/{path:path}", methods=["GET", "POST", "PUT", "DELETE"])
async def proxy_to_tool(tool_id: str, path: str, request: Request):
    """代理请求到工具子进程"""
    tool = registry.get(tool_id)
    if not tool or not tool.port:
        raise HTTPException(404, "Tool not running")

    async with httpx.AsyncClient() as client:
        resp = await client.request(
            method=request.method,
            url=f"http://127.0.0.1:{tool.port}/{path}",
            content=await request.body(),
            headers=dict(request.headers)
        )
        return Response(content=resp.content, status_code=resp.status_code)
```

async def startup():
